# 🍽️ Food Recipe App

A beautifully designed **Food Recipe App** built with **Flutter**.  
This app allows users to browse food categories, view meal details, mark favorite meals, and apply filters based on user preferences.

## 🚀 Features

- 🏠 **Home Screen:** Displays a list of food categories.  
- 🍛 **Meals Screen:** Shows all meals belonging to a selected category.  
- 📖 **Meal Detail Screen:** Displays detailed information about a meal, including:
  - Preparation time  
  - Complexity  
  - Ingredients  
  - Steps to prepare  
- ⭐ **Favorites Screen:** Lists meals marked as favorites.  
- 🔍 **Filter Screen:** Allows filtering meals by dietary preferences.

## 🧩 Tech Stack

- **Framework:** Flutter  
- **Language:** Dart  
- **Architecture:** Stateless & Stateful Widgets  
- **State Management:** Flutter Riverpod
